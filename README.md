# Sandhi in Nuclear Fusion: Recognizing Irreversible Commitment Prior to Ignition

**Author:** Uthraa Murali  
**Affiliation:** Independent Researcher, India  

---

## Abstract
This paper introduces *Sandhi* as an irreversible intermediate causal state that occurs prior to observable ignition in nuclear fusion systems. The work formalizes Sandhi as a point of irreversible commitment without realization, enabling early detection and intervention before catastrophic or uncontrollable outcomes.
